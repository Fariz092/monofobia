Folder ini digunakan untuk menyimpan aset statis seperti gambar (misalnya door.png). 

Tempatkan file aset Anda secara manual di sini (di dalam folder /public) saat Anda melakukan commit ke GitHub atau mengedit di repository lokal Anda. File di dalam folder ini dapat diakses secara langsung di aplikasi (contoh: '/door.png').
